<h2>一、项目简介</h2>
基于springboot的自习室管理系统：前端jsp、jquery、bootstrap、layui，后端 spring、mybatis，集成充值、桌子预约、学习时间统计、订单统计、订单管理等功能于一体的系统。

<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">
<h3 dir="auto" tabindex="-1">管理员</h3>
<ul dir="auto">
 	<li>用户管理：用户列表、根据登录名和姓名搜索、用户信息修改、用户信息删除</li>
 	<li>充值查询：充值列表、充值编号查询</li>
 	<li>桌子管理：桌子列表、根据桌号和状态搜索、桌子信息修改、桌子信息删除</li>
 	<li>订单管理：订单列表、根据订单编号和预约状态搜索、订单退订</li>
 	<li>系统设置：系统配置信息展示、信息修改（开始时间、结束时间、价格、店铺名、简介、地址）</li>
 	<li>订单统计：根据预定时间区间查询、折线图表展示</li>
</ul>
<h3 dir="auto" tabindex="-1"><a id="user-content-用户" class="anchor" tabindex="-1" href="https://github.com/YuLin-Coder/No45StudyRoomManagementSystem#%E7%94%A8%E6%88%B7" aria-hidden="true"></a>用户</h3>
<ul dir="auto">
 	<li>基本功能：登录、注册、退出、密码修改</li>
 	<li>个人中心：个人信息查询、个人信息修改、账户余额充值、充值记录查询、根据充值编号查询</li>
 	<li>店铺信息：店铺详情</li>
 	<li>桌子预约：桌子列表、空闲状态、点击预约、提交订单</li>
 	<li>我的订单：订单列表、根据订单编号和预约状态搜索、订单退订</li>
 	<li>学习时间统计：根据预约时间区间查询、折线图表展示</li>
</ul>
